# ⏰ Welcome 🖐 to the Clock with Time Zones Project

A **Digital Clock** that shows the current time in different time zones. This project uses JavaScript to manipulate the Date object and handle timezone calculations, displaying the time across various locations worldwide.

🌐 **Live Demo**: [Clock with Time Zones](https://atharvapatharkar.github.io/web-development-projects/Clock%20with%20Time%20Zones/index.html)

## 📂 Folder Structure

### 🔸 **Files Included**
- 📄 **`index.html`**: The main HTML file that structures the digital clock and time zone display.  
- 🎨 **`style.css`**: CSS file for styling the clock layout and time zone labels.  
- 🧑‍💻 **`script.js`**: JavaScript file to handle the Date object, time zone manipulation, and live updates of the time.  
- 📄 **`readme.md`**: Documentation for the project.  

---

## 🌟 Features

### 1️⃣ **Multiple Time Zones** 🌍  
   - 🌐 Shows the current time in various global time zones (e.g., New York, London, Tokyo).  
   - 🕒 Automatically updates every second.  

---

### 2️⃣ **Dynamic Date & Time** 📅  
   - 🗓️ The clock displays the current **date** and **time**, including **hours**, **minutes**, and **seconds**.  
   - ⏰ Uses JavaScript's **Date object** to get the accurate time and handle timezone conversions.

---

### 3️⃣ **User-Friendly Interface** 🎨  
   - 💻 **Responsive design** for mobile, tablet, and desktop views.  
   - 🌟 Clean and modern look with time zone names clearly displayed next to the respective clocks.  

---

## 🚀 How to Use

🛠️ Follow these steps to use the **Clock with Time Zones**:  
1️⃣ Open **`index.html`** in a browser.  
2️⃣ ⏰ View the **current time** in various time zones.  
3️⃣ 🔄 The clocks will automatically update every second for accurate time tracking.  
4️⃣ 📱 Resize your browser window to see the responsive layout in action.
