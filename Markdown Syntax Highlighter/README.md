# 🧮 Welcome to the Markdown Syntax Highlighter

This tool allows users to input markdown text and see the live preview with syntax highlighting for various markdown elements such as headings, lists, bold text, and more.

🌐 **Live Demo**: [Markdown Syntax Highlighter](https://atharvapatharkar.github.io/web-development-projects/Markdown%20Syntax%20Highlighter/index.html)

## 📂 Folder Structure

### 🔸 **Files Included**
- 📄 **`index.html`**: The main HTML file that holds the structure of the page.
- 🎨 **`style.css`**: CSS file to style the application with an attractive design.
- 🧑‍💻 **`script.js`**: JavaScript file responsible for live markdown preview and conversion.
- 🧑‍💻 **`readme.md`**: Project documentation for GitHub repository.

---

## 🌟 Features

### 1️⃣ **Live Markdown Preview** 🖥️  
   - 📝 Input markdown text and see the live preview with syntax highlighting.
   - 🌐 Supports headings, lists, bold, and italic text.

### 2️⃣ **Easy-to-Use Interface** ⚙️  
   - 🔄 Instant preview with no page reload.
   - ✍️ Simple and responsive design.

---

## 🚀 How to Use

🛠️ Follow these steps to use the Markdown Syntax Highlighter:  
1️⃣ Open **`index.html`** in a browser.  
2️⃣ Enter markdown text in the provided text area.  
3️⃣ The live preview will be updated instantly with the markdown syntax highlighting.
